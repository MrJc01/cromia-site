# Repositórios do GitHub (Crom & CromIA)

O código-fonte principal de nossas ferramentas, pacotes de compressão e utilitários de IA estão hospedados no GitHub sob o perfil do nosso engenheiro líder **[MrJc01](https://github.com/MrJc01)**.

---

## 🛠️ Ecossistema de Compressão Crompressor

Desenvolvemos algoritmos avançados focados em compressão inteligente de dados utilizando estruturas estatísticas e redes neuronais.

### 1. **[MrJc01/crompressor](https://github.com/MrJc01/crompressor)**
*   **Descrição**: O motor de compressão central (Core) da Crom. Implementa estruturas rápidas de empacotamento, otimizando o armazenamento e transferência de dados com foco em eficiência matemática.

### 2. **[MrJc01/crompressor-ia](https://github.com/MrJc01/crompressor-ia)**
*   **Descrição**: Módulo que introduz heurísticas de inteligência artificial aplicadas à decisão de blocos de compressão, melhorando as taxas de compressão em arquivos estruturados.

### 3. **[MrJc01/crompressor-neuronio](https://github.com/MrJc01/crompressor-neuronio)**
*   **Descrição**: Pesquisa avançada de compressão baseada em redes neurais artificiais. Explora o uso de pesos neurais compactados como funções matemáticas de aproximação para restaurar imagens, áudios e conjuntos de dados com perdas mínimas.

## 🧠 Motores de IA e Banco de Dados

Projetos dedicados ao processamento local de linguagem natural e utilitários de simulação de dados.

### 4. **[MrJc01/crom-microllm-think-vetor](https://github.com/MrJc01/crom-microllm-think-vetor)**
*   **Descrição**: Repositório contendo o código de inferência e treinamento do motor **Think Vetor**. Permite a execução local de micro-LLMs de raciocínio lógico (0.5B a 1B de parâmetros) utilizando tokens vetorizados e raciocínio latente para alta acurácia matemática.

### 5. **[MrJc01/crom-dbfakeai](https://github.com/MrJc01/crom-dbfakeai)**
*   **Descrição**: Biblioteca para modelagem e geração de bancos de dados sintéticos. Cria conjuntos de dados relacionais consistentes (com identidades fictícias lógicas, CPFs e transações válidas) preservando a distribuição de dados reais. Ideal para testes de estresse em bancos de dados sem expor informações sensíveis (LGPD).

### 6. **[MrJc01/crom-video-gen](https://github.com/MrJc01/crom-video-gen)**
*   **Descrição**: Utilitário experimental para geração e interpolação de frames de vídeo usando pequenos modelos de redes convolucionais e transformações matriciais rápidas.
